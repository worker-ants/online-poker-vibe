변경된 코드는 다음으로 구성됩니다:
- import 스타일 정리 (멀티라인 → 단일 라인) — 순수 스타일 변경

- `page.tsx`에 컴포넌트 마운트 추가 — 기존 소켓/상태 구조 변경 없음

### 요약

이번 변경은 동시성과 무관한 UI 컴포넌트 추가 및 import 정리 작업입니다. 공유 자원 접근, 비동기 처리, 상태 동기화와 관련된 코드 변경이 없으며, 신규 컴포넌트는 단순 로컬 상태만 사용하는 정적 표현형 컴포넌트입니다.
