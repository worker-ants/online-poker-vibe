## Overview

클라이언트의 페이지 구성, 컴포넌트 구조, 레이아웃을 정의합니다.

#### 구성 요소
- **닉네임 입력**: 현재 닉네임 표시 + 편집 가능. 미설정 시 설정 유도.
- **Room 만들기 버튼**: 모달 열림 (변형/모드/설정 선택)
- **Room 목록**: 실시간 업데이트. 각 항목에 방이름, 변형, 모드, 인원수, Join 버튼.
- **명예의 전당 링크**: Hall of Fame 페이지로 이동

### 2. 게임 화면 (`/game/[roomId]`)

#### 전체 레이아웃
```
┌────────────────────────────────────────────────────────┐
│  Top Nav: [닉네임]                        [? 도움말]   │
├─────────────────────────────────┬──────────────────────┤
│                                 │                      │
│         포커 테이블              │   플레이어 목록       │
│                                 │   ┌────────────────┐ │
│    [P2]     [P3]     [P4]       │   │ P1 ✓ Ready     │ │
│                                 │   │ P2 ✓ Ready     │ │
│  ┌─────────────────────┐       │   │ P3 ✗ Not Ready │ │
│  │  Community Cards     │       │   └────────────────┘ │
│  │  [?][?][?][ ][ ]    │       │                      │
│  │      Pot: 150        │       │   ┌────────────────┐ │
│  └─────────────────────┘       │   │ [준비 완료]     │ │
│                                 │   │ [게임 포기]     │ │
│    [P5]     [P1(me)]            │   └────────────────┘ │
│           [K♠][A♥]              │                      │
│                                 │   ┌────────────────┐ │
│  ┌─────────────────────┐       │   │ Fold │ Call     │ │
│  │ [Fold] [Call 20]     │       │   │ Raise [___]     │ │
│  │ [Raise ▲] [All-in]  │       │   │ All-in          │ │
│  └─────────────────────┘       │   └────────────────┘ │
│                                 │                      │
├─────────────────────────────────┴──────────────────────┤
└────────────────────────────────────────────────────────┘
```

#### 게임 시작 전 (Waiting)
- 플레이어 목록에 ready/not ready 상태 표시
- "준비 완료" 버튼 (토글)
- Host: "추방" 버튼 (각 Guest 옆)
- 모든 플레이어 ready + 2명 이상 시 자동 시작

#### 게임 진행 중 (Playing)
- 포커 테이블: 플레이어 좌석, 커뮤니티 카드 (Hold'em), 팟
- 자신의 카드: 핸드 카드 표시
- 베팅 컨트롤: 자기 턴일 때만 활성화
- 턴 타이머: 현재 턴 플레이어에게 카운트다운 표시
- "게임 포기" 버튼: 확인 다이얼로그 후 퇴장

#### 변형별 테이블 차이
| 요소 | Texas Hold'em | 5 Card Draw | 7 Card Stud |
|------|-------------|-------------|-------------|
| 커뮤니티 카드 | 0~5장 | 없음 | 없음 |
| 플레이어 카드 | 2장(비공개) | 5장(비공개) | 3~7장(공개/비공개 혼합) |
| 특수 UI | - | Draw 교환 선택 | 공개카드 별도 표시 |

## State Management

### React Context (변경 빈도 낮음)
- **SocketProvider**: Socket.IO 인스턴스, 연결 상태
- **IdentityProvider**: playerId, nickname, setNickname
- **ToastProvider**: 알림 큐

### Zustand Store (변경 빈도 높음)
- **gameStore**: GameState, RoomList, connectionStatus
- 컴포넌트별 slice 구독으로 불필요한 리렌더 방지

## Card Rendering

- CSS/HTML + 유니코드 심볼 기반 (이미지 불필요)
- 무늬: ♠(spades), ♥(hearts), ♦(diamonds), ♣(clubs)
- 색상: 빨강(hearts, diamonds), 검정(spades, clubs)
- 앞뒤 전환: CSS `transform: rotateY()` 애니메이션
- 카드 크기: aspect-ratio 2.5:3.5, 반응형

## 도움말 모달

변형별로 다른 규칙 설명 표시:
- 게임 진행 순서
- 핸드 랭킹 표
- 베팅 액션 설명
- 변형 고유 규칙 (Draw의 교환, Stud의 공개/비공개 등)
